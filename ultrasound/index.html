reference available for recorded clips

Recorded clips carry the real nerve mask and needle tip/tail from the dataset. For a live volunteer with no dataset answer key, the instructor places a manual reference mark. The reference is the answer key; the cobalt and amber model overlay is only a second opinion.

Freeze a frame, then start the self test. You commit your call before the model or the answer key is revealed.

--

You are scored against the real dataset ground truth, never against the model. The cobalt and amber overlay is a fallible second opinion; when it disagrees with the answer key, or reads confident on an incoherent image, that is the lesson.

The gate reads three raw-frame input-quality features only (no probability map), so the in-browser decision reproduces the verified Python decision exactly. It rejects NON-ANATOMICAL input (noise, blank, dropout). A true in-domain ultrasound frame with no nerve in view is UNTESTABLE here because the Nerve-UTP dataset contains no nerve-absent frame; that limitation is stated honestly and is not covered by these numbers.
